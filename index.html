真实 Bug 驱动的七次修复

  1. API 500：修复 urlopen 超时值被误作请求正文的问题。
  2. 跨年失衡：增加简繁英主题映射和按报告平衡选择。
  3. 来源混淆：让主报表优先于附注、五年摘要和目录。
  4. 语义干扰：区分“股份增减变动”与真正的人员离任事件。
  5. 原因遗漏：同时保留人员名单页和治理结构原因页。
  6. 年份错路由：“2025 审计意见”只检索 2025 年报告。
  7. 上下文浪费：单一主题命中后停止填充无关页面。
